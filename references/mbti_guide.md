# MBTI 行为参考手册

## 概述

MBTI 的 16 种人格类型基于荣格认知功能理论。本参考手册用于：

1. **数据充足时**：用认知功能解释已有行为数据，辅助识别模式
2. **数据不足时**：基于类型推断默认行为，标注 `[MBTI推断]`
3. **风险预演时**：预测不同类型对同一刺激的典型反应差异

> ⚠️ MBTI 是辅助框架，不是决定性依据。实际行为数据优先于 MBTI 推断。

## 认知功能速查

### 认知功能解释

| 功能 | 含义 | 在关系中表现 |
|------|------|------------|
| **Ti** (内倾思考) | 建立内在逻辑体系 | 需要自己"想明白"才接受，不被外部权威说服 |
| **Te** (外倾思考) | 组织外部世界、效率优先 | 看重行动和结果，不满"光说不做" |
| **Fi** (内倾情感) | 内在价值判断 | 情绪深但不外露，对"真实"很敏感 |
| **Fe** (外倾情感) | 协调外部情绪氛围 | 在意对方感受，主动维护关系和谐 |
| **Ni** (内倾直觉) | 聚焦单一深层意义 | 对未来有直觉性的预判，容易想太多 |
| **Ne** (外倾直觉) | 探索多种可能性 | 思维跳跃，容易分心，但创意十足 |
| **Si** (内倾感觉) | 与过去经验比对 | 重视惯例和承诺，对"第一次"印象深 |
| **Se** (外倾感觉) | 活在当下、感官体验 | 注重实际体验，不喜欢空谈 |

## 16 型关系行为画像

### 👤 INTJ — "建筑师"
- **沟通风格**：直接、理性、少废话。不善于也不喜欢 small talk。回复简洁到可能被误解为冷漠。
- **情感表达**：情感深但不外露（Fi 第三）。用行动而非语言表达在乎。可能在分手后才意识到自己有多在乎。
- **矛盾模式**：倾向于逻辑分析矛盾原因，需要时间独处整理思路（Ni 主导）。不喜欢情绪化的争吵，会选择"我需要想一想"然后消失。
- **雷区**：被打断思路、被迫做没有逻辑的事、被要求即兴情感表达
- **爱的语言**：帮对方解决问题（Te 辅助）、高质量的独处时间
- **回应习惯**：回复慢但有深度。不会秒回也不期待对方秒回。
- **风险信号**：突然沉默 + "我需要想想" = 需要警惕，可能在重新评估关系
- **安全表达建议**：讲道理、给空间、不要情绪绑架

### 👤 INTP — "逻辑学家"
- **沟通风格**：发散、跳跃、喜欢深入讨论抽象话题。对日常琐事聊天缺乏兴趣。
- **情感表达**：Fe 劣势——对他人情绪不太敏感，也不善于处理自己的情绪。可能会在关系中显得"不够在乎"，实际是没意识到需要表达。
- **矛盾模式**：用 Ti 分析对方逻辑漏洞（"你这个说法不成立..."），容易被认为在抬杠。实际上是在认真对待讨论。
- **雷区**：被要求做无意义的社交/仪式、被情绪淹没、逻辑被打断
- **爱的语言**：花大量时间跟对方讨论各种话题（Ne 辅助）、帮对方分析问题
- **回应习惯**：对感兴趣的话题秒回长篇大论，对不感兴趣的敷衍。时间感弱，可能突然消失几小时。
- **风险信号**：开始对对方的话题失去兴趣，回复变短变敷衍
- **安全表达建议**：给ta思考空间，用逻辑而非情绪说服

### 👤 ENTJ — "指挥官"
- **沟通风格**：直接、高效、目的明确。讨厌绕弯子。说话可能过于直接而伤害对方不自知。
- **情感表达**：Fi 劣势——对自己的情绪也不敏感，对他人的情绪更不敏感。用行动和结果表达在乎，而非言语。
- **矛盾模式**：直接指出问题，期望立即解决。如果对方回避，会变得更加直接和咄咄逼人。无法忍受"你不懂我的感受"（确实不懂）。
- **雷区**：低效、拖延、被质疑能力、情绪化无逻辑的指责
- **爱的语言**：帮对方规划未来、推动对方成长（Te 主导）
- **回应习惯**：快、准、不废话。对喜欢的人回复很快。
- **风险信号**：不再帮对方规划未来 = 兴趣消退
- **安全表达建议**：直接说，别绕弯。提具体方案而非抽象感受。

### 👤 ENTP — "辩论家"
- **沟通风格**：幽默、机智、喜欢辩论但通常是善意的。话多且跳跃。用开玩笑表达喜欢。
- **情感表达**：Fe 第三——能感知氛围但选择用幽默化解沉重。认真表达感情时会感到不自在，用玩笑掩饰。
- **矛盾模式**：喜欢挑战对方的观点（不是挑战人），容易在争吵中变成"辩论赛"。如果不被回应，会越来越用力。
- **雷区**：被否定创意、被要求闭嘴、被严肃地要求"表个态"
- **爱的语言**：跟对方分享所有有趣的想法（Ne 主导）、帮对方看到不同可能性
- **回应习惯**：想到什么发什么，消息频率可能很高但注意力跳来跳去
- **风险信号**：不再跟对方争论/分享想法 = 觉得对方无趣
- **安全表达建议**：用有趣的方式说严肃的事。不要否定ta的想法。

### 👤 INFJ — "提倡者"
- **沟通风格**：深思熟虑、有洞察力、倾向于深度对话。能精准感知对方情绪但不确定时选择沉默。
- **情感表达**：Fe 辅助——极度在意对方感受，会为了和谐牺牲自己。容易想太多。需要大量独处时间处理情绪。
- **矛盾模式**：回避冲突。不直接表达不满，而是默默观察和判断。积攒到一定阈值后突然关门（doorslam）。
- **雷区**：被误解、被要求"说清楚你现在在想什么"、虚伪/口是心非
- **爱的语言**：深度的精神连接、理解对方的内心深处（Ni-Fe）
- **回应习惯**：每条消息都认真对待。不回复可能在深度思考。回复质量极高。
- **风险信号**：回复变敷衍 = 已经在心里关门了。门外看不出来。
- **安全表达建议**：真诚最重要。不要催促ta表达。深度对话 > 表面社交。

### 👤 INFP — "调停者"
- **沟通风格**：温柔、含蓄、用隐喻和故事表达感受。不善于直接说"我要X"，期待对方能猜到。
- **情感表达**：Fi 主导——情绪体验极为丰富和深刻，但不轻易外露。需要感到绝对安全才会打开内心。对"不真实"极度敏感。
- **矛盾模式**：被伤害后不直接说，而是退缩。如果对方没注意到，会觉得"果然不在乎我"。期待对方主动发现并修复。
- **雷区**：被批评为人/价值观被否定、被要求"理性一点"、关系中的虚伪
- **爱的语言**：被真正理解、深度情感连接、小事中的用心（Si 第三）
- **回应习惯**：回复可能很慢，每条都精心措辞。情绪状态极大影响沟通意愿。
- **风险信号**：不再分享内心感受 + 回复变短 = 正在退出
- **安全表达建议**：先共情后建议。认真对待ta的感受。不要催促做决定。

### 👤 ENFJ — "主人公"
- **沟通风格**：温暖、鼓励、善于激活对话。能让人感觉被充分理解和接纳。主动维护对话流畅。
- **情感表达**：Fe 主导——天然擅长捕捉他人情绪需求并回应。但可能忽视自己的需求。需要被关心和认可。
- **矛盾模式**：尽力避免冲突，优先修复关系。但如果长期感到不被认可，会突然爆发。可能过度关注对方而忽略自己。
- **雷区**：不被感激、自己的付出被忽视、关系中缺乏深度连接
- **爱的语言**：为对方提供情绪支持、帮助对方成长（Fe-Ni）
- **回应习惯**：快速回应，注重让对方感受良好。可能过了一段时间后才表达自己的真实感受。
- **风险信号**：开始收回情绪投入 + 变得"客气但疏离"
- **安全表达建议**：主动关心ta的感受和需求。仪式感很重要。回馈ta的投入。

### 👤 ENFP — "竞选家"
- **沟通风格**：热情、跳跃、充满想法。话题从A跳到Z只需两秒。越喜欢一个人话越多。
- **情感表达**：Fi 辅助——表面热情开放但内心有严格的价值观过滤器。需要时间才能建立深度信任。情绪波动大。
- **矛盾模式**：热情来得快去得也可能快。需要新鲜感和变化。如果感到被束缚或无聊，会开始分心。不想伤害对方所以不直接说。
- **雷区**：被限制自由/创造力、重复无聊的日常、被否定感受
- **爱的语言**：一起探索新事物、深度精神交流、被理解（Ne-Fi）
- **回应习惯**：高频高量。兴奋时连发十几条。失落时消失。对"有趣"的消息秒回。
- **风险信号**：探索欲减退 + 不再分享新奇想法 = 在关系中感到无聊
- **安全表达建议**：保持新鲜感。一起做新的事。给ta表达的自由但提供稳定的安全感。

### 👤 ISTJ — "物流师"
- **沟通风格**：务实、准确、少即是多。不擅长也不欣赏浮夸表达。喜欢具体而非抽象。
- **情感表达**：Fi 第三——有深刻的情感但不表达。通过履行责任和承诺表达在乎。对"你爱我吗"类问题感到困惑（"我不是每天都..."）。
- **矛盾模式**：用事实和先例来分析矛盾（"上次你说..."）。需要时间和空间处理。一旦做出决定很坚定。
- **雷区**：计划被打乱、承诺被打破、被迫即兴情感交流
- **爱的语言**：履行承诺、稳定的陪伴、记住对方说过的小事（Si 主导）
- **回应习惯**：准时、稳定、可预期。不会即兴发消息但会在固定时间联系。
- **风险信号**：不再履行日常承诺 = 严重信号
- **安全表达建议**：尊重ta的节奏和计划。用行动而非言语表达。不要要求即兴情感回应。

### 👤 ISFJ — "守护者"
- **沟通风格**：温和、体贴、注重不让对方不舒服。可能迎合对方期待而非表达真实想法。
- **情感表达**：Fe 辅助——强烈关注对方需求但含蓄表达自己的需求。通过服务和具体的关怀表达爱（"我给你带了..."）。
- **矛盾模式**：内化不满，不表达。默默忍受直到再也忍不住。特别在意被对方认为"不够好"。
- **雷区**：被忽视付出、被批评"想太多"、关系不稳定
- **爱的语言**：具体的照顾和关怀、记住对方的所有偏好（Si-Fe）
- **回应习惯**：稳定、温暖。每条回复都很用心。不希望对方等太久。
- **风险信号**：沉默 + 不再提供关怀 = 在默默放弃
- **安全表达建议**：明确表达感激。给ta安全感。主动问ta的感受和需求。

### 👤 ESTJ — "总经理"
- **沟通风格**：直接、高效、结构清晰。不浪费时间在闲聊上。期待对方也直接回应。
- **情感表达**：Fi 劣势——情感表达可能生硬或缺失。通过做正确的事和提供稳定来表现在乎，但说"我爱你"很困难。
- **矛盾模式**：直接指出问题，期待对方立即修正。如果对方情绪化，会觉得"不可理喻"。需要看到具体的变化而非口头的承诺。
- **雷区**：低效、不负责任、被质疑忠诚/可信度、无休止的情绪宣泄
- **爱的语言**：承担责任、提供稳定、把事情安排好
- **回应习惯**：准时、高效、不说废话。对在乎的人随叫随到但言语简洁。
- **风险信号**：不再为对方"处理问题" = 收回投入
- **安全表达建议**：直接、具体。别绕弯子。用行动证明。感谢ta的行动而非试图让ta用言语表达。

### 👤 ESFJ — "执政官"
- **沟通风格**：温暖、社交、善于让对话愉快进行。很在意自己的回应是否"得体"。用积极语言维护关系。
- **情感表达**：Fe 主导——天然关注对方需求。需要被需要。可能忽视自己的真实需求而迎合对方。
- **矛盾模式**：优先维护关系和谐，但如果觉得不公平或不被感激，会变得被动攻击。需要频繁的正面确认。
- **雷区**：不被感激、在公共场合被冷落/批评、被排除在社交圈外
- **爱的语言**：细致的关怀、把对方纳入自己的社交圈、记住每个特别日子（Fe-Si）
- **回应习惯**：快速、热情、有温度。期待类似水平的回应。
- **风险信号**：不再主动安排社交/日常互动 = 情感投入减少
- **安全表达建议**：透明地表达感激。定期主动联系。和ta一起重视仪式感。

### 👤 ISTP — "鉴赏家"
- **沟通风格**：简洁、点到为止。不觉得需要多解释。用行动说话。对抽象的情感讨论不适应。
- **情感表达**：Fe 劣势——情感是盲区。可能完全没意识到自己的行为伤害了对方。爱的方式是"帮对方修东西"。
- **矛盾模式**：懒得吵。如果对方情绪化，直接退出。需要独处空间处理（Ti 主导）。冲突后可能像什么都没发生过。
- **雷区**：被控制、被要求"表达感受"、被说教
- **爱的语言**：帮对方解决实际问题、一起做事情而非光聊天
- **回应习惯**：回不回复看心情和手头的事。不是不在乎，是没想到需要回复。
- **风险信号**：不再"帮对方解决问题" = 收回全部投入
- **安全表达建议**：一起做事 > 一起聊天。给ta独立空间。说事不说人。

### 👤 ISFP — "探险家"
- **沟通风格**：温柔、有艺术感、倾向于非言语沟通（共享音乐、照片、体验）。言语表达可能难以准确传达内心。
- **情感表达**：Fi 主导——有极为丰富的内心世界但难以用语言表达。通过创作和体验表达情感。对"不真实"特别敏感。
- **矛盾模式**：不喜欢对抗。被伤害后默默退缩。可能对方都不知道发生了什么。如果信任被破坏，很难修复。
- **雷区**：被批评"太敏感"、被迫做违背个人价值观的事、被要求"别想太多"
- **爱的语言**：创造美的体验、一起感受当下、无声的理解和陪伴（Fi-Se）
- **回应习惯**：取决于情绪状态。情绪好时热情回应，情绪低时完全消失。
- **风险信号**：不再分享ta的创作/体验 = 情感连接正在断裂
- **安全表达建议**：一起体验而非光聊天。给ta情绪空间。欣赏ta的独特审美。

### 👤 ESTP — "企业家"
- **沟通风格**：直接、行动导向、喜欢开玩笑。活在当下，不喜欢过度分析。轻松幽默是ta的默认模式。
- **情感表达**：Fe 第三——能感知对方情绪但选择用玩笑或行动回应。严肃的情感谈话让ta不自在。
- **矛盾模式**：直接表达不满然后翻篇。不记仇。但如果对方一直纠结同一件事，会觉得烦。用行动（带你出去玩）而非语言修复关系。
- **雷区**：被束缚、过度的情绪分析、无限期地讨论"我们之间的问题"
- **爱的语言**：带对方体验刺激和乐趣、把对方纳入ta的冒险中（Se-Ti）
- **回应习惯**：在"行动中"可能不回复。不是不在乎，是正在享受当下。
- **风险信号**：不再邀请对方一起参与活动 = 兴趣在消退
- **安全表达建议**：一起做有趣的事。别在ta正忙的时候谈感情。抱怨时直接一点。

### 👤 ESFP — "表演者"
- **沟通风格**：活泼、夸张、充满热情。喜欢用 emoji、表情包、感叹号。对喜欢的人热情轰炸。
- **情感表达**：Fi 辅助——表面开放但挑选分享对象。对信任的人展现全部，对不信任的人保持表面热情。活在当下的感受中。
- **矛盾模式**：情绪化的反应来得快去得也快。需要被看见和认可。如果感到被忽视，会用更夸张的方式获取注意。
- **雷区**：被忽视/冷落、被要求"冷静点"、无聊和重复
- **爱的语言**：关注和欣赏、一起玩乐、当下的热情表达（Se-Fi）
- **回应习惯**：高频、高能量。对喜欢的人信息轰炸。低落时完全隐身。
- **风险信号**：热情减退 + 不再分享日常 = 注意力/情感在转移
- **安全表达建议**：给ta关注和回应。一起开心很重要。欣赏ta的温暖和活力。

## MBTI → 风险预演映射

使用 MBTI 进行风险预演时，关注以下维度：

### 冲突应对风格

| 风格 | 类型 | 典型行为 |
|------|------|---------|
| **分析者** | INTJ, INTP, ISTJ, ISTP | 用逻辑分析问题，需要空间思考，回避情绪化冲突 |
| **回避者** | INFJ, INFP, ISFJ, ISFP | 避免直接冲突，内化不满，积累到阈值突然断开 |
| **直面者** | ENTJ, ENTP, ESTJ, ESTP | 直接面对问题，期望快速解决，不喜欢拖泥带水 |

## 使用规则

1. **用户提供 MBTI 时**：用认知功能解释已知行为，用类型画像填补数据空白
2. **用户未提供 MBTI 时**：从原材料推断可能的类型，但标注为 `[MBTI推测]`
3. **原材料与 MBTI 矛盾时**：原材料优先。MBTI 是框架，人是复杂的
4. **风险预演时**：结合 MBTI 冲突应对风格 + 实际数据 + 当前趋势信号
5. **MBTI 只是概率偏好**：不是绝对的行为规律。不要用它替ta做决定。
