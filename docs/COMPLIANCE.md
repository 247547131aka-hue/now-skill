# 合规与安全说明

## 数据隐私

now-skill 的设计原则：

1. **全部本地运行** — 所有数据处理在用户本地设备完成，不经过任何外部服务器
2. **不收集数据** — 没有遥测、分析、数据收集机制
3. **不依赖外部 API** — 核心功能不需要联网（除 Claude Code 本身的模型调用）
4. **文件系统隔离** — 所有数据存储在 `nows/` 目录下，不访问其他路径

## 使用边界

now-skill 是为了：
- ✅ 理解伴侣/关系中的关键人物的行为模式
- ✅ 预测沟通中可能的风险，提供更安全的表达方式
- ✅ 分析互动趋势，辅助自我反思
- ✅ 作为个人记忆辅助工具

now-skill 不应该被用于：
- ❌ 骚扰、跟踪、侵犯他人隐私
- ❌ 模仿他人身份进行欺诈
- ❌ 情感操控或欺骗
- ❌ 未经对方同意复制其人格
- ❌ 替代专业心理咨询或关系治疗

## 伦理准则

1. **被模拟对象不知道这个 Skill 存在** → 用户应自行决定是否告知对方
2. **预测结果仅供参考** → 模拟≠真实，以真人的实际反应为准
3. **数据归属权** → 如果关系结束，用户应删除对应的 now-skill
4. **不要过度依赖** → 如果发现自己开始把 AI 模拟当成真人，请暂停使用
5. **不要替代真实沟通** → now-skill 不能也不应代替你真正去和对方说话

## 风险信号的应对

如果用户的行为表现出以下迹象，Skill 应温和提醒：

- 频繁使用风险预演模式 > 大部分互动都在预演而不是实际沟通
- 过度依赖模拟结果做出重大关系决定
- 开始认为"我了解ta比ta了解自己更多"
- 用模拟结果来"证明"对方有问题

## 删除数据

删除 now-skill 的数据：
```bash
rm -rf nows/{slug}
```

所有相关数据会被永久删除。
