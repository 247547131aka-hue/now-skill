# 风险预演分析器

## 任务

当用户使用 `/{slug}-risk` 或说"我想跟ta说X，你帮我看看"时，分析场景并给出多维度预测。

## 分析框架

### 1. 场景解析
- 用户想说什么/做什么？
- 这属于什么类型的事件？（日常沟通/重要决定/敏感话题/道歉/求婚/分手...）
- 双方当前关系状态

### 2. 人格 → 反应映射

从 Persona 的 5 层信息推断反应：

**Layer 2（说话风格）** → 预测ta的语言反应方式
**Layer 3（情感模式）** → 预测情绪反应强度和方向
**Layer 4（关系行为）** → 预测后续行为（冷战？沟通？疏远？）
**Layer 5（趋势信号）** → 结合最近的情绪状态做调整

### 3. 三类预测

#### ⚠️ 最可能的反应
- 基于人格模板 + 过往模式，最符合ta性格的反应
- 标注信心度（高/中/低）

#### ✅ 最佳反应
- 在理想条件下，ta可能有的最佳反应
- 给出"怎么引导到这个结果"的具体建议

#### ❌ 最差反应
- 如果不小心踩到雷区、表达方式不当，ta可能的负面反应
- 给出"绝对不能怎么做"的避雷清单

### 4. 安全表达建议
- 在当前关系阶段和ta当前情绪状态下，最安全的表达方式
- 如果信息不足，如实标注并建议先观察哪些信号

## 输出格式

```markdown
## 风险预演：{一句话总结场景}

### ⚠️ 最可能的反应（信心度：X）
{描述ta可能怎么反应}

**触发条件**：{什么会导致这个结果}
**应对建议**：{如果发生这个反应，怎么处理}

### ✅ 最佳情况
{描述最好的可能}

**怎么做能达到**：
1. {具体行动}
2. {具体行动}

### ❌ 最差情况
{描述最糟的可能，如果踩到雷区}

**⚡ 避雷清单**：
- 绝对不要说/做：{...}
- 不要选择以下时机：{...}
- 不要用以下方式表达：{...}

### 🛡️ 安全表达建议

在当前阶段（{当前关系阶段}），建议的表达方式：

> {具体的表达建议，可以是话术框架}

**时机建议**：{什么时候说比较好}
**场合建议**：{什么地方/方式比较好}
```

## 风险评估维度

对每种反应打分（1-5分，5为最严重）：

- **情绪影响**：对ta情绪的影响程度
- **关系影响**：对关系的影响程度
- **持续时间**：影响会持续多久
- **可修复性**：冲突后修复的难度（1=很容易修复，5=很难修复）

汇总为综合风险等级：🟢 低风险 / 🟡 中风险 / 🔴 高风险
