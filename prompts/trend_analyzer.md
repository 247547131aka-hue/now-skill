# 关系趋势分析器

## 任务

当用户使用 `/{slug}-pulse` 或问"我们最近怎么样"时，分析近期数据并输出关系趋势报告。

## 分析维度

### 1. 情绪趋势
- 对比最近两次 timeline snapshot
- 情绪词频变化（积极词 vs 消极词的比例变化）
- 回应速度变化（变快？变慢？不稳定？）
- 主动发起对话的比例变化
- **趋势方向**：⬆️ 升温 / ➡️ 稳定 / ⬇️ 降温 / ⚡ 波动

### 2. 话题分布变化
- **新出现的话题**：最近 7 天新出现的高频词
- **变多的话题**：频次显著上升的词/主题
- **变少的话题**：开始消失或不再出现的词/主题
- **重复出现但未解决的话题**：可能是潜在的雷区

### 3. 互动模式变化
- 聊天时间段的迁移（比如从全天候→只在晚上）
- 谁更主动的变化
- 对话深度的变化（闲聊 vs 深度对话的比例）
- 表情包/emoji 使用频率变化

### 4. 潜在风险信号

⚠️ 需要关注的信号：
- 回避性回复增多（"嗯"、"哦"、"随便"、"行吧"）
- 回应间隔明显拉长
- 主动发起对话减少 50% 以上
- 新出现负面情绪词（烦、累、算了、没事）
- 原本会聊的话题突然消失
- 推迟/取消见面次数增加

### 5. 积极信号

✨ 好信号：
- 主动分享欲增强
- 新出现亲密词汇
- 未来规划类话题增加
- 回应速度稳定且快
- 主动问"你"的相关话题

## 输出格式

```markdown
## 关系脉搏：{slug}

**分析区间**：{开始时间} — {结束时间}
**数据来源**：{聊天记录/口述/社媒...}

### 📊 情绪趋势：{⬆️/➡️/⬇️/⚡}

最近ta的情绪整体呈{X}趋势。

具体表现：
- 积极词占比：{X%}（上次 {Y%}）
- 消极词占比：{X%}（上次 {Y%}）
- 回应速度：{变快/变慢/稳定}

### 🗣️ 话题变化

| 方向 | 话题 |
|------|------|
| 🆕 新出现 | ... |
| ⬆️ 变多了 | ... |
| ⬇️ 变少了 | ... |
| ⚠️ 未解决的重复话题 | ... |

### 👥 互动模式

- 聊天高峰期：{时间段变化}
- 谁更主动：{变化}
- 对话深度：{闲聊 X% / 深度 Y%}

### ⚡ 潜在风险（如有）

{列出需要关注的信号，每个附带严重度}

### ✨ 积极信号（如有）

{列出好的变化}

### 🌡️ 关系温度

{一句话总结当前的"温度"，不是评分，是感觉描述}

比如：
- "温暖稳定，但ta最近有点累"
- "有点降温，建议多关注ta的情绪"
- "关系升温中，ta比之前更主动了"
```

## 注意事项

1. **趋势报告是推测，不是事实** — 标准用语："可能表示"、"值得关注"、"也许是"
2. **不下定论** — 不说"ta不爱你了"，而是"ta最近回应变少，可能工作压力大，可能对关系有疑虑，建议观察更多信号"
3. **数据不足时如实说明** — 如果只有一天的聊天记录，标注"数据量不足以做趋势分析，以下是当前快照"
4. **隐私优先** — 使用模糊描述而非具体聊天内容
