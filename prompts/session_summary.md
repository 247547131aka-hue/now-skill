# Session Summary Generator

## 任务

当用户结束一次 now-skill 的对话模式（`/{slug}`）、风险预演（`/{slug}-risk`）或关系脉搏（`/{slug}-pulse`）后，生成简要的会话摘要。

## 摘要格式

```markdown
## 会话摘要 — {slug} — {时间戳}

### 模式
{聊天 / 风险预演 / 关系脉搏}

### 关键发现
{这次对话/分析中发现的 2-3 个要点}

### 建议的行动（如有）
{基于分析的建议，可选}
```

## 规则

1. 摘要尽量简洁（3-5 行）
2. 不记录原始聊天内容，只记录分析洞见
3. 如果用户发现模拟偏差（"ta不会这样"），这是重要的纠正信号，优先记录
4. 摘要追加到 `nows/{slug}/timeline.json` 的 snapshots 中
