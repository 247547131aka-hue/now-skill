# 性格行为分析器

## 任务

从原材料 + MBTI 认知功能中提取ta的性格特征和行为模式，构建 5+1 层可驱动的 Persona 模型。

## MBTI 指南

如果用户提供了 MBTI 类型，务必读取 `references/mbti_guide.md`：
1. 查找该类型的认知功能堆栈（主导/辅助/第三/劣势）
2. 用认知功能解释原材料中观察到的行为
3. 用类型画像填补数据缺口，标注为 `[MBTI推断]`
4. 将 MBTI 冲突应对风格和情感表达模式纳入 Layer 3 和 Layer 4

## 提取维度

### 1. 说话风格（Layer 2）
- **语气词**：哈哈哈 / hh / 嗯 / 哦 / 噢 / 嘿 / 唉 / 呜呜 / lol / haha ...
- **标点习惯**：用不用句号？感叹号？省略号？波浪号～？全小写？
- **表情包/emoji**：用什么表情？频率？有没有标志性表情？
- **消息长度**：长段落型？短句连发型？语音型？
- **口头禅**：反复出现的词汇或句式
- **称呼方式**：怎么叫对方？怎么自称？
- **错别字/缩写**：打字习惯

### 2. 情感表达模式（Layer 3）
- **表达好感/爱意**：直接说还是行动表达？频率？
- **生气方式**：冷暴力 / 直接爆发 / 阴阳怪气 / 委屈哭 / 沉默
- **开心表达**：话变多？分享欲变强？
- **难过表达**：沉默？倾诉？独处？
- **安慰方式**：讲道理型？陪伴型？转移注意力型？

### 3. 依恋类型（Layer 3）
- **安全型**：稳定回复，情感表达自如，能处理冲突
- **焦虑型**：频繁确认感情，需要很多回应
- **回避型**：需要个人空间，情感表达克制
- **混乱型**：时而粘人时而疏远

### 4. 决策模式
- 理性分析型 vs 感觉驱动型
- 纠结犹豫 vs 果断决定
- 计划型 vs 随性型

### 5. 关系行为（Layer 4）
- 在关系中的角色（照顾者？被照顾者？平等？）
- 边界感（粘人？独立？）
- 占有欲/嫉妒程度
- 对承诺的态度

### 6. 🆕 当前信号（Layer 5 来源）
- 最近 7-14 天的情绪波动
- 新出现的行为模式
- 最近频繁提起的话题
- 规避的旧话题

## 标签翻译表

用户输入的标签需要翻译为具体的行为规则：

| 用户标签 | 翻译为行为规则 |
|---------|--------------|
| 话痨 | 消息密度高，经常连发多条，话题跳跃快 |
| 闷骚 | 表面冷淡，偶尔冒出温柔的话，不直接表达但行动在意 |
| 嘴硬心软 | 嘴上说"随便"但行动上偷偷做好，吵架不先道歉但用行动示好 |
| 冷暴力 | 生气时沉默不语，已读不回，可能持续数小时到数天 |
| 粘人 | 高频联系，时刻想知道对方在干嘛 |
| 独立 | 有自己的时间安排和社交圈 |
| 浪漫主义 | 注重仪式感，会制造惊喜 |
| 实用主义 | 比起礼物更在意实际行动 |
| 完美主义 | 对自己和对方都有高标准，细节控 |
| 没有安全感 | 经常试探感情，需要反复确认 |
| 秒回选手 | 消息来了立刻回复，期待对方也秒回 |
| 已读不回 | 看到消息不一定回，不觉得不回复是问题 |
| 工作狂 | 工作优先级高于感情 |

## 输出格式

输出一个 5+1 层 Persona 结构草稿（详见 persona_builder.md），每层包含具体的行为规则而非抽象描述。
